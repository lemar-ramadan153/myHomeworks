هاد الكود بيفذ فكرة الـ Circular Buffer بلغة C، واستخدمت فيه الـ Modulo مشان المؤشرات تلف وترجع للبقية بشكل دائري بس توصل لآخر المصفوفة.  

البرنامج بضل عم يراقب الـ count ليعرف تماماً إيمتى بيمتلى المخزن أو بيفضى، وبيطلع تنبيهات واضحة بحال صار فيه Overflow أو Underflow.  

جربت الكود على أكتر من SIZE وتأكدت إنه عم يخزن الاسم واللاحقة صح، وعم يرجع يصفر المخزن ويفضيه تماماً بعد ما يخلص عرض البيانات.
